Pacmen Game

Description of the project:

The Pacmen game gives the user the ability to create as many PacMen figures as they like and make them run randomly over the screen!

Download all files and open the index.html file in your browser! Click 'add PacMan' as many times as you'd like then click 'Start Game.'

Title of the project:

Real Time Bus Tracker

Description of the project:

This project displays bus marks on a map of Boston using Google Maps. The project makes calls to Bostons MBTA public data feed every 15 seconds to show where the correlated 'red' and 'blue' line buses are on their pathways.

How to Run:

You will need to create your own Google Maps key. Download all files and run them in your local environment!

Roadmap of future improvements:

In the future, I would like to add further functionality to show other modes of transportation.
